# Boas-vindas ao repositório do projeto Trybetunes!
* Neste projeto foi usada a API Itunes para fazer uma interface capaz de reproduzir trechos de músicas. A organização é em Artistas, albúms e músicas.

## Funcionalidades:

* Pesquisar artistas

* Escolher entre albums

* Escolher músicas

* Tocar e pausar músicas

* Pesquisar albums

* Pesquisar músicas
